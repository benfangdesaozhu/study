#### 扩展运算符

扩展运算符（spread）是三个点（...）。
```
console.log(...[1,2,3]) // 1 2 3

console.log(1, ...[1,2,3]) // 1 1 2 3

...[1,2] // Uncaught SyntaxError: Unexpected token '...'(报错)
```

扩展运算符主要应用于函数调用。
```
...value运用在函数中的时候，只能放在最后一个参数（或者只有一个参数。它的后面不能有其他参数，有的话会报错）

var array = []
function b(...value) {
    array.push(...value);
}
b(1,2,3) 
// array = [1,2,3]
```
